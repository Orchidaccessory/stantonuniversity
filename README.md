# stanton university
The redesigned site
addresses weaknesses in the original page, such as static event listings, poor visual hierarchy, and
limited interactivity. Key features include hero imagery, interactive event cards, color-coded tags,
quick tabs, in-page registration, and responsive design. Mobile interface optimization is
recommended to align with student digital habits and improve accessibility.
Visual Design
The current page relies heavily on text, lacks engaging visuals, and presents events in
long, difficult-to-scan lists. Event types are not clearly differentiated, making browsing
inefficient.
User Experience
The site is not optimized for mobile devices. Event registration opens external pages,
disrupting the user flow. Filtering and sorting options are absent, requiring users to scroll
extensively.
Accessibility
Small fonts and low contrast make reading challenging for users with visual impairments.
Mobile navigation is limited, affecting usability on smartphones and tablets.
Functionality
Static listings, lack of interactive features, and absence of in-page registration reduce
engagement and efficiency for users.
